Technique Discussion

As stated before hand, there are many differrent types of Dark UX patterns that we see all of the time. Other forms of Dark UX patterns that were not discussed before include Nagging, Sneaking, and Interface Interference.

Nagging is a form of Dark UX that is defined as minor redirecitons of functionality that continues over multiple interatctions. An example of this is the app "Instagram" asking for the location of the user without having a direct no button, rather saying that the interaction does not have to happen immidately.

Sneaking is defined as attempting to hide, disguise, or delay information that has relevance to the user. This can come in forms such as continuing subscriptions like console's internet connection whenever the user purchases their online plan, and the user does not have to cancel the subscription if they feel that they want to continue or it does not feel as though the subscription is paid so far in advance, the user forgets when the subscription is charged.

Interface interference is defined as manipulation to the user interface that requires some action over others. This would make the user feel as though that they have to take action while at the same time hiding other options from the user that they could pursue. An example of this would be the large ads Yahoo uses to distract the user from updating security preferences.
